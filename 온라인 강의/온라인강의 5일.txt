분류에 추가, 수정, 삭제 등의 변화가 일어 나는지 여부에 따라 결정
분류에 변화가 한 번이라도 있으면 테이블에 저장해서 작업해야 함
분류별로 검색 및 통계 등의 작업이 이루어져야 하므로 배열에 저장시 분류 코드 등의 값을 지정해 놓아야 함
테이블에 저장하면 속도가 느린 단점이 있다.

SI : System Integration
시스템 통합을 의미하는 용어이나 요새는 보통 파견업체 등을 의미하는 용도로도 사용됨

세계최초의 컴퓨터 : aniac
최초의 상용컴퓨터 : advac

orderList : 하나의 주문에 대한 정보들(주문자정보, 수취인정보, 결제관련 정보 등)
orderDetail : 해당 주문에서 주문한 상품(들)에 대한 정보
하나의 테이블에 저장하면 하나의 주문으로 여러 가지의 상품을 구매했을 경우 동일한 정보(주문자정보, 수취인정보, 결제관련 정보 등)가 반복되어 들어가게 됨
동일한 정보가 반복되어 레코드를 채우는 것을 DB는 지양함
여러 레코드가 되게 하는 상품에 대한 정보를 밖으로 빼서 새로운 테이블로 만든 뒤 기존의 테이블과 1 대 n 의 관계를 설정하는 것을 정규화라 함
단, 동일한 과목이라도 시간이나 내용이 조금 다를 수 있음
