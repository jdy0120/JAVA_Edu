자바 스크립트 (Java Script)
 - 웹 페이지에 동적인 움직임을 부여하는 언어로 자바와는 상관없음
 - 인터프리터 언어, 변수의 자료형을 별도로 선언하지 않음
 - 기본적으로 자바의 문법과 유사하며, 대소문자를 구분함

자바 스크립트의 역할
 - HTML요소 내용을 변경할 수 있음
 - HTML요소 속성을 변경할 수 있음
 - HTML요소 스타일을 변경할 수 있음
 - HTML요소를 숨기거나 보이게 할 수 있음

자바 스크립트와 HTML문서의 연결법
 - 인라인 방식 : HTML태그의 이벤트 속성에서 직접 입력
 - 내부 파일방식 : HTML문서에서 <script>태그 내부에서 입력
 - 외부 파일방식 : 따로 ~.js로 자바 스크립트 파일을 생성하여 HTML문서에서 불러들여 사용

변수
 - 특정 값 또는 객체를 저장하기 위한 메모리 상의 공간
 - 변수 선언시 자료형을 사용하지 않고 'var'라는 키워드로 선언함
 - 변수명 규칙 : 자바와 동일하며, 한글로 이름을 지어도 됨

자바 스크립트의 자료형
 - 기본자료형 : number(정수 & 실수), string(문자열), boolean(논리갑스 부울형)
 - 복합자료형 : object(객체), array(배열), function(함수)
 - 특수자료형 : undefined(정의되지 않은 상태로 값이 정해지지 않은 상태), null(널)

연산자
산술 연산자 : +, - * / %
 - 피연산자 : 숫자, 결과값 : 숫자

증감연산자 : ++ --
 - 피연산자 : 숫자, 결과값 : 숫자
 - 단항연산자로 해당 숫자값을 1씩 증가 또는 감소 시킴
 - prefix 또는 posfix의 여부에 따라 값이 1씩 차이날 수 있음


대입 연산자 : = += -= *= /= %=
 - 값을 변수에 넣는 연산자로 복합연산자를 사용하면 연산 후 변수에 넣어줌

비교연산자 : == != > >= < <=
 - 피연산자 : == != : 모든 값 / > >= < <= : 숫자, 결과값 : 논리값

논리연산자 : && || !
 - 피연산자 : 논리값(조건), 결과값 : 논리값
 - 조건이 여러 개일 경우 하나의 논리값 결과로 만들기 위한 연산자

조건연산자 : (식) ? 참 : 거짓
 - 삼항연산자라고도 하며, 식이 참인지 여부에 따라 다른 작업을 함
 - if문으로 완벽하게 대체되므로 사용빈도가 높지는 않음

제어문
 - 프로그램은 기본적으로 위에서 아래로 실행되지만, 경우에 따라 특정 위치로 이동(분기)하거나 실행문들을 반복해서 실행시켜야 할 때 사용하는 기능
 - 제어문에는 분기문과 반복문의 두 가지가 있다.

분기문
 - 조건을 주어 조건이 참인지 여부에 따ㅓ라 실행할 실행문들이 달라지는 제어문
 - if문과 switch문이 있음

1. if문
 - 대표적인 분기문으로 가장 많이 사용되는 제어문
 - 문법
if (조건식) {
   실행문(들);
}
[
else if (조건식2) {
   실행문(들)2;      
}...
else if (조건식n) {
   실행문(들)n;
}
]

 - 실행문이 한 줄일 경우 중괄호를 생략할 수 있음
 - else문이 잇으면 실행문들 중 하나를 반드시 실행함
    else문이 없으면 조건에 따라 실행문들 중 하나도 실행하지 않을 수도 있음

2. switch문
 - 분기문이나 if문과는 다르게 조건으로 판단하지 않고, 특정 값과의 비교로 판단하여 실행
 - if문으로 완벽하게 대체되므로 사용하지 않아도 프로그램에는 큰 영향이 없음
 - 문법
switch (값) {
   case 비교값1 :
      실행문(들);
      break;
   ...
   case 비교값n :
      실행문(들)n;
      break;
   default :
      실행문(들);
}
- case문을 한 줄에 여러개 입력할 경우 각 case별 값을 ㅐㄱfh anRdjwna
   예) case 1: case2: => 1이거나 2이면

반복문
 - 조건을 주어 조건이 참인동안 지정한 실행문을 반복하여 샐행시키는 제어문

1. for문
 - 숫자값으로 초기화한 후 특정 숫자값까지 반복하는 반복문으로 배열등에서 많이 사용됨
