글자 관련 CSS 명령

CSS에서 글꼴을 지정할 때는 font-family 속성을 사용하여 원하는 글꼴을 지정할 수 있다. 이 때 글꼴명을 복수로 지정할 수 있는데 쉼표로 구분하며 입력 순서가 지정 우선 순위가 된다.

글꼴을 복수로 지정하는 이유는 사용자의 시스템에 해당 글꼴이 없을 경우를 대비할 수 있지만 이 또한 없을 경우를 위해 마지막에 'serif', 'sans-serif', 'cursive', 'fantasy', 'monospace' 중 하는 지정한다.


문자색을 black, white 등의 영문 색상명으로 지정할 수 있다.

문자색을 6자리의 16진수로 RGB형식으로 표현할 수 있다.

문자색을 총 9자리의 10진수로 RGB형식으로 표현할 수 있다.

문자색을 백분율로 표현할 수 있다.
