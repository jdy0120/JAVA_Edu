JQuery 요소 탐색 메소드
 - html요소를 다른 요소와의 관계를 기반으로 찾기 또는 선택하는데 사용되는 메소드들
 - JQuery를 이용하여 DOM을 탐색한다.

1. 조상 탐색
 - 조상은 부모, 조부모, 증조부모 등을 의미함
 - parent(filter) : 선택된 요소의 직접적인 부모 요소를 리턴
	filter : 여러 부모 요소들 중 특정 요소(들)를 지정하기 위한 선택자가 사용됨
	 - 쉼표로 구분하여 여러 개의 선택자를 사용할 수도 잇음
 - parents([filter]) : 선택된 요소의 모든 조상 요소를 리턴
 	 - 쉼표로 구분하여 여러 개의 선택자를 사용할 수도 잇음
 - parentsUntil(top,filet) : 2개의 주어진 매개변수 사이의 모든 조상 요소를 리턴
	top : 선택한 요소부터 top으로 지정된 요소까지를 선택해줌
	filter : 여러 부모 요소들 중 특정 요소(들)를 지정하기 위한 선택자가 사용됨
	 - 쉼표로 구분하여 여러 개의 선택자를 사용할 수도 잇음
 - offsetParent() : 최초로 위치하는 부모 요소를 리턴
 - closest() : 선택된 요소의 첫번째 조상을 리턴
